### Introduction

This project is the source code for Suraj Kumar Choudhary's personal website, presenting his skills, projects, and contact details. The website is built using HTML, CSS, and JavaScript.

### Project Structure

The project structure is organized as follows:

- **assets**: Contains subdirectories for images, styles, and scripts used in the project.
  - `./assets/images`: Images used in the project.
  - `./assets/css`: Custom styles for the website.
  - `./assets/js`: Custom JavaScript code.

- **index.html**: The main HTML file containing the structure of the website.

### Features

1. **Navigation Bar**: Interactive navigation bar with links to different sections of the website.

2. **Hero Section**: Introduction section with a brief overview, photo, and project statistics.

3. **About Section**: Details about Suraj's education and background, including relevant icons.

4. **Portfolio Section**: Showcase of Suraj's project works with links to view each project.

5. **Skills Section**: Display of Suraj's skills with progress bars.

6. **Contact Section**: A form for visitors to drop a message and contact information.

7. **Social Icons**: Links to Suraj's social media profiles in the footer.

8. **Back to Top Button**: Convenient link to navigate back to the top of the page.

### Usage

To view the website locally, simply open the `index.html` file in a web browser. Make sure to have an active internet connection to load external styles and scripts.
